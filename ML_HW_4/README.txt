###################################
# Instructions for Running the Code
###################################

1. Requirements:
   - Python 3.8 or above.
   - Install dependencies using:
     pip install numpy matplotlib

2. Files and Usage:
   - CNNHw4.ipynb: Contains the Q-learning implementation.
   - nelsonML.ipynb: Focuses on comparing Boltzmann exploration strategies.

3. Running Instructions:
   - Open the notebooks using Jupyter Notebook or JupyterLab.
   - For Epsilon-Greedy exploration:
     * Set epsilon values to 0.1, 0.2, and 0.3.
     * Run the notebook and check the output for convergence episodes.
   - For Boltzmann exploration:
     * Modify temperature decay settings in the notebook.
     * Run the notebook and observe convergence.

4. Notes:
   - Ensure all required libraries are installed.
   - Analyze plots and outputs for insights.
